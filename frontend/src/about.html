Sobre Nuestro Proyecto

Descubriendo el valor de las viviendas en Andalucía con Machine Learning.

Nuestra Misión

Este proyecto nace con el objetivo de proporcionar una herramienta intuitiva y precisa para la estimación de precios de viviendas en la comunidad autónoma de Andalucía. Utilizando técnicas avanzadas de recolección de datos, preprocesamiento y modelado con Machine Learning, buscamos ofrecer transparencia y conocimiento accesible tanto para compradores y vendedores como para profesionales del sector inmobiliario.

Fases Clave del Proyecto

1. Recolección y Adquisición de Datos

Extracción de miles de anuncios de viviendas de portales inmobiliarios líderes, capturando características esenciales como superficie, número de habitaciones, baños, ubicación geográfica (latitud, longitud) y tipo de propiedad.

2. Preprocesamiento y Limpieza

Tratamiento de valores atípicos, imputación de datos faltantes, y transformación de variables categóricas (como 'tipo de propiedad') mediante técnicas como One-Hot Encoding para preparar los datos para el modelo.

3. Ingeniería de Características

Creación de nuevas características relevantes a partir de las existentes para mejorar la capacidad predictiva del modelo. Aunque en este proyecto nos centramos en las características directas, esta fase es crucial en muchos problemas de ML.

4. Selección y Entrenamiento del Modelo

Evaluación de diversos algoritmos de regresión. Se seleccionó un modelo Gradient Boosting (como XGBoost o LightGBM) por su robustez y alto rendimiento, entrenándolo con el conjunto de datos procesado.

5. Evaluación y Despliegue

Validación del modelo utilizando métricas como el Error Absoluto Medio (MAE) y R². Finalmente, despliegue del modelo a través de una API RESTful (Flask) para ser consumida por esta aplicación web.
