Modelo Predictivo de Precios de Vivienda en Andaluca (con Coordenadas Geogrficas)

Este cuaderno de Jupyter tiene como objetivo desarrollar un modelo predictivo para la estimacin de precios de viviendas en Andaluca. A diferencia de enfoques anteriores, este modelo utilizar directamente las coordenadas de latitud y longitud como caractersticas predictivas, en lugar de la caracterstica derivada 'provincia'. Se explorarn tcnicas de preprocesamiento de datos, clustering opcional basado en geolocalizacin y diversos algoritmas de regresin.

Pasos del Desarrollo:

  1. Carga de Libreras y Datos.
  2. Preprocesamiento de Datos (Incluyendo Latitud/Longitud, Excluyendo Provincia).
  3. Implementacin de Clustering para Segmentacin de Mercado (Opcional, usando Coordenadas Geogrficas).
  4. Divisin de Datos para Modelado Predictivo.
  5. Definicin, Entrenamiento y Evaluacin Inicial de Mltiples Modelos Predictivos.
  6. Optimizacin de Hiperparmetros del Mejor Modelo.
  7. Validacin Cruzada y Seleccin del Modelo Final.
  8. Evaluacin Detallada del Modelo Final en el Conjunto de Prueba.
  9. Anlisis de Importancia de Caractersticas del Modelo Final (con Latitud/Longitud).
  10. Guardado del Modelo Final para Despliegue.

2. Preprocesamiento de Datos (Incluyendo Latitud/Longitud, Excluyendo Provincia)

Identificaremos las caractersticas numricas (superficie, habitaciones, baos, latitud, longitud) y categricas (tipo_propiedad). La caracterstica provincia ser excluida. Crearemos transformadores para la imputacin de valores faltantes (mediana para numricos, moda para categricos), escalado para caractersticas numricas (StandardScaler), y codificacin para caractersticas categricas (OneHotEncoder). Estos pasos se combinarn usando ColumnTransformer. Finalmente, prepararemos X (caractersticas) e y (variable objetivo precio).
